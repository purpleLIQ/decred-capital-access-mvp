# User Guide

This demo shows how a Decred holder could borrow against DCR without selling it.

## Core Idea

The borrower locks DCR in a Decred escrow. The app tracks collateral, loan health, repayment, and release. In a future mainnet alpha, the borrower would receive USDC or another liquid asset while keeping exposure to DCR.

## Demo Mode

Demo mode uses seeded loans and simulated chain events. It is safe to click through because it does not use real money or private keys.

## Screens

- Quote: price a potential DCR-backed loan.
- Status: inspect collateral, LTV, escrow, and activity.
- Repay: simulate repayment and collateral release.
- Operator: simulate admin actions and liquidation review.
- Market: inspect price and liquidity assumptions.
- Docs: review sources and trust assumptions.
