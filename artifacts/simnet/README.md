# Simnet Proof Artifacts

This folder is reserved for local simnet proof outputs.

Expected files after a collateral-release proof run:

```text
check-config.json
probe-rpc.json
escrow-utxos.json
unsigned-release-preview.json
validate-artifacts.json
```

Do not commit private keys, seeds, wallet passphrases, wallet dumps, or production credentials.

Proof artifacts should only include safe JSON outputs from the simnet proof scripts.
